## Run Locally

Clone the project

```bash
  git clone https://github.com/Your Username/Todo-List-Component
```

Go to the project directory

```bash
  cd Todo-List
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
